### Python Verifier

`tools/verify_vectors.py` independently verifies all vector files by:

- validating JSON schema structure
- decoding `hex` and `repeat_byte` input encodings
- recomputing `CoreHash` and `PhoneticHash` from `stream_hex`
- asserting all four mandatory anchors are present
- validating error object fields per `ErrorKind`

This is the project's first **cross-language hash verification**:
Python independently reproduces the same hashes as Rust on all 128 ok-vectors.

## Pending Corrections (Future Spec Revision)

### CR-08 (Proposed)

**Issue:** GT-S02 (اللَّه) in Spec §9 shows expected stream with LAM+SUKUN
on first LAM, but the input contains no U+0652 (SUKUN) codepoint.

**Resolution in this implementation:** No SUKUN is synthesized. Output is
fully determined by input (Axiom A1).

**Recommended spec fix:** Update GT-S02 stream and hashes to match the input,
OR add U+0652 to the normative input bytes.
